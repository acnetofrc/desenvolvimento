Pipelining

O grande gargalo da execução de uma instrução é a busca desta instrução. As instruções são buscadas e armazenadas no prefetch buffer a pipeline utiliza esta mesma estratégia, mas de forma mais enfática. As instruções são buscadas e armazenadas no prefetch buffer a pipeline utiliza esta mesma estratégia, mas de forma mais enfática.
